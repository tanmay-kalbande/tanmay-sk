# Project-Scoped Rules

## Design Aesthetics & Styling Philosophy
- **V5 Landing Page Authority**: For any UI modifications or new features, strictly take design inspiration and styling patterns from the V5 landing page (`LandingPage.tsx`) and portfolio components.
- **Color Palettes**: Stick strictly to the defined color variables in `landing.css`.
  - Light mode uses `#f5f5f3` backgrounds, `#ffffff` surfaces, `#1a1a1a` text, and `#c8451a` accent values.
  - Dark mode uses `#0e0e10` backgrounds, `#1c1c1b` surfaces, `#f0ede8` text, and `#e05a35` accent values.
- **Typography & Details**: Use editorial typography stacks (like Cormorant Garamond for display headers and Roboto Mono for metadata tags/toggles) and avoid custom, ad-hoc inline styled layout elements. Keep all changes clean, minimal, and fully theme-integrated.
